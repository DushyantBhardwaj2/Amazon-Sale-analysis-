# Amazon Sales Analysis - Capstone Project 4

## Project Overview
This project focuses on analyzing large-scale Amazon sales data using Python and Streamlit. It demonstrates the ability to handle e-commerce datasets to uncover insights into sales trends, regional performance, and product profitability.

The project is mapped to **Unit IV: Distributed Databases** and AI analytics, simulating a distributed architecture (Kafka -> Spark -> Delta -> Cassandra) for a robust data pipeline.

## ⚡ Key Features
- **Sales Overview:** Real-time KPI tracking for Revenue, Profit, and Margins.
- **Business Analytics:** Answers critical business questions like highest profit regions, unit sales trends, and YoY growth.
- **Regional Insights:** Deep dive into regional and country-specific performance.
- **Architecture Story:** Visualization of a distributed data pipeline for production-scale AI deployment.

## 🛠️ Technology Stack
- **Frontend:** Streamlit
- **Data Processing:** Pandas (Simulating Spark DataFrame operations)
- **Visualization:** Plotly, Graphviz
- **Deployment:** Streamlit Cloud

## 🚀 How to Run Locally
1. Clone the repository.
2. Navigate to the project folder:
   ```bash
   cd Amazon-Sale-analysis-
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```
